==================
saved_trajectories
==================
Holds two folders containing saved trajectories for UR5s for predetermined paths. 

.. figure:: saved_traj_folder.png
   :align: center

   The two folders.

The UR5_1 folder contains saved trajectories for picking up individual packages from the shelf and placing it on the conveyor, whereas the UR5_2 folder has saved trajectories for placing a package into it's corresponding coloured bin from a common home position.
